# Product Principles

## Principle 1: Zero Friction Start
- 어떤 입력도 없이 RUN 가능
- 준비 부족으로 시작을 막지 않는다

## Principle 2: User is Always Moving
- 멈춤도 하나의 상태다
- 기록되지 않는 시간은 없다

## Principle 3: No Retrospective Editing
- 기록은 수정할 수 없다
- 하루는 되돌릴 수 없다

## Principle 4: Visual First, Numbers Later
- 숫자는 보조 정보
- 흐름과 움직임이 1차 정보
